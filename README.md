# Mechanic Quotes

## Frontend codebase for Mechanic Quotes


This Project is built with:
- React
- MUI
- Vite(build tool)



To run the app, go to your terminal and:
- clone the repository
- run `npm install`
- run `npm run dev`
- open the resulting url in your browser or http://localhost:5173
